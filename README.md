# Local Civitai Video Wall v2

本地 Civitai 风格短视频浏览工具，适合 720×1280、1–3MB、约 6 秒的 MP4 素材。

## v2 新增

- 路径默认留空
- 支持手动输入路径
- 支持点击按钮弹出 Windows 文件夹选择框
- 支持“记住路径”
- 支持“扫描子文件夹”
- 支持 4 / 5 / 6 / 7 / 8 / 9 列切换
- 支持同时播放上限：12 / 18 / 24 / 30
- 支持沉浸模式
- 顶部工具栏更紧凑
- 保留搜索、排序、随机、暂停、点击大窗播放、打开所在位置

## 使用方法

1. 解压 zip。
2. 双击 `start.bat`。
3. 浏览器打开：

```text
http://127.0.0.1:8787
```

## 路径说明

第一次打开时，路径默认留空。

你可以手动输入路径，例如：

```text
F:\Users\Downloads\kenpechi\MP4
```

也可以点击“选择文件夹”，弹出 Windows 文件夹选择框。

然后点击“扫描”。

## 记住路径

如果勾选“记住路径”，工具会把路径保存到 `config.json`，下次打开自动加载。

如果不勾选，关闭后不会保存路径，下次仍然为空。

## 扫描子文件夹

默认不扫描子文件夹。

勾选“扫描子文件夹”后，会递归扫描当前目录下所有子目录的视频。

支持格式：`.mp4` / `.webm` / `.mov` / `.m4v`

## 推荐设置

普通浏览：6列，播放上限24。

看细节：4列或5列，播放上限12或18。

快速扫素材：8列或9列，播放上限24或30。

## 注意

瀑布流里的视频为了符合浏览器自动播放规则，默认是静音播放。点击视频后打开的大播放器可以手动开声音。
