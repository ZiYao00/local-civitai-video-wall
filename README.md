# Local Civitai Video Wall

一个本地版 Civitai 风格短视频瀑布流浏览工具。

它适合用来快速浏览本地短视频素材，尤其是竖屏、短时长、数量较多的本地视频。页面会自动播放当前屏幕内可见的视频，并在视频滚出屏幕后暂停或释放资源，避免一次性加载全部视频。

## 功能特点

- 本地视频文件夹扫描
- 支持手动输入视频目录
- 支持 Windows 文件夹选择框
- 支持记住路径
- 支持扫描子文件夹
- 支持 4 / 5 / 6 / 7 / 8 / 9 列切换
- 支持同时播放上限：12 / 18 / 24 / 30
- 当前屏幕内视频自动静音循环播放
- 滚出屏幕后自动暂停，离开较远后释放资源
- 支持搜索文件名
- 支持排序：修改时间、文件名、文件大小
- 支持随机浏览
- 支持沉浸模式
- 点击视频后可大窗口播放
- 支持打开视频所在位置

## 使用场景

这个工具适合：

- 浏览 Civitai 下载的视频素材
- 浏览 AI 生成的短视频
- 浏览 ComfyUI / Stable Diffusion / Wan / Kling / Runway 等工具生成的视频
- 快速筛选大量本地短视频
- 像看动态素材墙一样浏览本地视频文件夹

## 运行环境

- Windows 10 / Windows 11
- Python 3.9 或更高版本
- 推荐使用 Chrome / Edge 浏览器

## 支持格式

目前支持：

```text
.mp4
.webm
.mov
.m4v
```

## 使用方法

### 1. 启动工具

双击：

```text
start.bat
```

或者在命令行运行：

```bash
python app.py
```

### 2. 打开浏览器

访问：

```text
http://127.0.0.1:8787
```

### 3. 选择视频目录

你可以：

- 手动输入视频文件夹路径
- 点击“选择文件夹”按钮
- 勾选“记住路径”
- 勾选“扫描子文件夹”

然后点击“扫描”。

## 推荐设置

普通浏览：

```text
6列
播放上限 24
```

看细节：

```text
4列 或 5列
播放上限 12 或 18
```

快速扫素材：

```text
8列 或 9列
播放上限 24 或 30
```

## 注意事项

浏览器自动播放视频通常要求视频静音，所以瀑布流里的视频默认是静音播放。

点击视频后，会打开大窗口播放器，你可以在播放器里手动开启声音。

## 配置文件

工具会在本地使用：

```text
config.json
```

保存你的设置，例如：

- 是否记住路径
- 上次使用的视频目录
- 是否扫描子文件夹
- 默认列数
- 播放上限
- 排序方式
- 是否开启沉浸模式

如果你要上传到 GitHub，建议不要上传自己的 `config.json`，只保留：

```text
config.example.json
```
