# Local Civitai Video Wall

[English README](README.md)

一个本地版 Civitai 风格短视频瀑布流浏览工具。

![预览](assets/preview01.gif)

它适合用来快速浏览本地短视频素材，尤其是 AI 生成视频。页面会自动播放当前屏幕内可见的视频，并在视频滚出屏幕后暂停或释放资源，避免一次性加载全部视频。

## 功能特点

- 本地视频文件夹扫描
- 支持图片 + 视频混合浏览
- 支持从图片弹窗进入全屏图片幻灯片
- 支持手动输入视频目录，并提供单层文件夹路径自动补全
- 支持新版 Windows 文件夹选择框
- 支持记住路径
- 支持带内存缓存的左侧分栏侧边栏浏览磁盘、最近路径和收藏路径
- 支持扫描所选目录下最多 2 层子文件夹
- 可自行添加文件名排除关键词，并选择仅图片或图片和视频；默认关键词为 `fanart`、`thumb`
- 支持 4 / 5 / 6 / 7 / 8 / 9 / 12 / 16 列切换
- 支持同时播放上限：6 / 8 / 12 / 18 / 24 / 30
- 当前屏幕内视频自动静音循环播放
- 滚出屏幕后自动暂停，离开较远后释放资源；翻页、重扫、关闭弹窗时也会释放旧媒体资源
- 图片墙只加载接近可见区域的图片，远离后释放资源，不生成磁盘缩略图缓存
- 500MB 以上视频默认显示轻量占位封面，可在设置中允许墙内预览大文件
- 大型扫描结果每页渲染 120 个卡片，避免一次创建上千个媒体元素
- 显示轻量性能统计，包括扫描耗时、渲染耗时、本页数量和已加载媒体数
- 扫描数量与审核状态显示在媒体网格上方右侧
- 支持搜索文件名
- 支持按文件大小和最近修改时间筛选
- 支持排序：修改时间、文件名、文件大小
- 支持随机浏览
- 支持导出当前审核列表为 CSV
- 支持沉浸模式
- 支持暗色 / 亮色主题切换
- 支持小、标准、大三档界面字体
- 调整浏览器窗口宽度时，左侧文件夹分栏与媒体网格保持独立，不再互相覆盖
- 点击图片后可浮层大窗口预览，并支持上一张/下一张、鼠标滚轮切换、全屏幻灯片和隐藏/显示控制
- 点击视频后可浮层大窗口播放，并支持上一段/下一段、播放模式、滚轮音量、全屏和隐藏/显示控制
- 支持打开视频所在位置
- 支持安全移动到 `_video_wall_review` 或 Windows 系统回收站，并可选择是否确认
- 视频预览支持单视频循环、顺序播放和随机播放
- 幻灯片支持播放/暂停、上一张/下一张、1-15 秒间隔、效果、Fit、循环、全屏和显示/隐藏控制栏
- 幻灯片效果支持无效果、淡入淡出、滑动、动态漂移和随机
- 默认英文界面，顶部工具栏可切换中文
- 支持收藏、精选标记和审核筛选

## 使用场景

这个工具适合：

- 浏览 Civitai 下载的视频素材
- 浏览 AI 生成的短视频
- 浏览 ComfyUI / Stable Diffusion / Wan / Kling / Runway 等工具生成的视频
- 快速筛选大量本地短视频
- 像看动态素材墙一样浏览本地视频文件夹

## 运行环境

- Windows 10 / Windows 11
- Python 3.10 或更高版本，推荐 Python 3.12
- 推荐使用 Chrome / Edge 浏览器

这个工具会以本地 Python 服务运行。如果电脑没有安装 Python，`start.bat` 和 `service.bat` 不能直接启动工具。

## 使用方法

### 1. 启动工具

手动模式：

双击：

```text
start.bat
```

浏览器会自动打开，并显示命令行窗口。关闭这个命令行窗口就等于退出工具。

服务模式：

```text
service.bat
```

这个文件会打开英文菜单，可以选择后台启动、停止或重启后台服务、安装开机自启、取消开机自启、打开浏览器、后台启动并打开浏览器或检查状态。开机启动会使用 Windows APPDATA 目录下的稳定辅助脚本，不再依赖临时目录。

也可以在命令行运行：

```bash
python app.py
```

然后访问：

```text
http://127.0.0.1:8787
```

### 2. 选择视频目录

你可以：

- 手动输入视频文件夹路径，例如 `C:\Users\你的用户名\Videos`
- 点击侧边栏按钮，打开左侧分栏文件夹树
- 使用路径栏内部的星标按钮收藏当前路径
- 使用路径栏内部的历史按钮打开最近路径或清空历史记录
- 点击“Choose Folder / 选择文件夹”按钮
- 勾选“Remember path / 记住路径”
- 勾选“Scan subfolders / 扫描子文件夹”

从文件夹面板或 Windows 文件夹选择框选中路径后，会自动填入路径栏并开始扫描。手动输入路径时，可以按 Enter 或点击“Scan / 扫描”。

### 3. 浏览图片和视频

- 点击图片可以打开图片预览窗口。
- 图片预览里可以使用左右按钮、键盘方向键或鼠标滚轮切换上一张/下一张。
- 点击“幻灯片（全屏）”可以进入图片幻灯片模式。
- 幻灯片模式下，可以在顶部工具栏控制播放/暂停、间隔、效果、Fit、循环和完整屏幕。
- 点击视频可以打开视频预览窗口。
- 视频预览里可以使用左右按钮切换上一段/下一段。
- 视频预览里可以使用鼠标滚轮调整音量。
- 视频播放模式支持单视频循环、顺序播放和随机播放。
- 想专注观看图片、幻灯片或视频时，可以使用“隐藏控制”。

### 4. 界面选项

顶部工具栏和设置菜单提供这些常用控制：

- 语言：英文 / 中文
- 主题：暗色 / 亮色
- 路径历史：可清空最近路径，收藏路径会保留
- 收藏路径和历史记录都支持单条移除
- 文件名排除规则：最多添加 30 个关键词，不区分大小写，可随时启用、停用或删除
- 大视频墙内预览：500MB 以上默认关闭，但点击占位封面仍可按需打开播放

## 推荐设置

普通浏览：

```text
6列
播放上限 24
```

看细节：

```text
4列 或 5列
播放上限 12 或 18
```

快速扫素材：

```text
8列 / 9列 / 12列 / 16列
播放上限 24 或 30
```

## 注意事项

浏览器自动播放视频通常要求视频静音，所以瀑布流里的视频默认是静音播放。500MB 以上的视频默认在墙内显示占位封面，避免误加载电影或演唱会等超大文件；可在设置中修改。

点击视频后，会打开大窗口播放器，你可以在播放器里手动开启声音。

## 配置文件

工具会在本地使用：

```text
config.json
```

保存你的设置，例如路径、列数、播放上限、排序方式、沉浸模式、语言、主题和按钮样式。

幻灯片的间隔、效果、Fit 模式和循环设置也会保存在 `config.json`。

这个文件可能包含你的本地路径，不建议上传到 GitHub。上传仓库时保留 `config.example.json` 即可。
